# Step 03: Internationalization (i18n)

## Objective
Set up i18next with Bulgarian and English translations.

## Tasks

### 1. Create i18n Configuration
Create `src/i18n/i18n.js`:

```js
import i18n from 'i18next';
import { initReactI18next } from 'react-i18next';
import bg from './bg.json';
import en from './en.json';

i18n
  .use(initReactI18next)
  .init({
    resources: {
      bg: { translation: bg },
      en: { translation: en }
    },
    lng: 'bg', // default language
    fallbackLng: 'bg',
    interpolation: {
      escapeValue: false
    }
  });

export default i18n;
```

### 4. Initialize i18n in App
Update `src/main.jsx` to import i18n:

```jsx
import React from 'react';
import ReactDOM from 'react-dom/client';
import App from './App.jsx';
import './index.css';
import './i18n/i18n.js'; // Add this line

ReactDOM.createRoot(document.getElementById('root')).render(
  <React.StrictMode>
    <App />
  </React.StrictMode>,
);
```

## Expected Outcome
- i18n configured with bg/en translations
- All UI text translatable
- Language toggle working
- Default language: Bulgarian
